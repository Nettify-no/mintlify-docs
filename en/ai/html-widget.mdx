## What is an HTML widget?

An HTML widget lets you embed content from other services directly on your website. Many tools you already use — like Calendly, Google Maps, Tidycal or Trustpilot — give you an HTML code you can paste to display their content on your site.

<Info>
  You don't need to know how to code. All you need is to copy the code from the service you use and tell Nettify AI where you want it.
</Info>

---

## How to add an HTML widget

<Steps>
  <Step title="Find the embed code from your service">
    Most services have an **"Embed"** or **"Share"** button that gives you an HTML code. Copy this code.
  </Step>

  <Step title="Tell Nettify AI what you want to do">
    Paste the code in the chat and describe where you want the widget.
  </Step>

  <Step title="Review the result">
    Nettify AI places the widget on your page. Check that it displays correctly in the preview.
  </Step>
</Steps>

## Ready-made prompts for popular widgets

### Calendly booking widget

<Tip>
  **Copy this prompt and paste it into Nettify AI:**
</Tip>

```
Add a Calendly booking widget to the contact section on the homepage.
Use this code:
<iframe src="https://calendly.com/[your-calendly-url]/30min" width="100%" height="700" frameborder="0"></iframe>

Place the widget in its own section with:
- Heading: "Book a Meeting With Us"
- Short intro: "Pick a time that works for you and we'll have a no-obligation chat."
- Design: centered, with good padding around the widget. Clean background.
```

### Google Maps map

<Tip>
  **Copy this prompt and paste it into Nettify AI:**
</Tip>

```
Embed a Google Maps map showing our address: [your address].
Place the map in the contact section with:
- Heading: "Find Us"
- Full-width map, 400px height
- Below the map: address, phone number and a "Get directions" link
- Design: map with rounded corners and light shadow. Responsive on mobile.

Use this code:
<iframe src="https://www.google.com/maps/embed?pb=[your-embed-code]" width="100%" height="400" frameborder="0" allowfullscreen></iframe>
```
