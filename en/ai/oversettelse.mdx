## Reach customers in multiple languages

Do you run a business with international customers, tourists or multilingual staff? With Nettify AI you can easily add a language switcher to your website, so visitors can switch between languages as needed.

---

## Ready-made prompts for multilingual websites

### Add language switcher with flags

<Tip>
  **Copy this prompt and paste it into Nettify AI:**
</Tip>

```
Add a language switcher in the navigation at the top right with a clean and subtle design.

The language switcher should:
- Show two options: English (EN) with a US/UK flag, and [other language] ([code]) with its flag
- The active language should be highlighted
- When clicking a language, all text on the page should switch to the selected language
- Design: small flag icons with language code beside them. Dropdown or inline buttons. Matches the rest of the navigation.

Translate all content on the homepage to [other language]:
- Menu, headings, body text, buttons, footer — everything should be translated
- Keep company name, addresses and phone numbers as they are
- Use professional and natural [other language] — not a direct translation
```

---

### Translate the entire website

<Tip>
  **Copy this prompt and paste it into Nettify AI:**
</Tip>

```
Translate our entire website to [target language]. This includes:
- All pages: homepage, services, about us, contact and [other pages]
- Navigation menu and submenus
- Button text and link text
- Footer with contact info and social media links
- All headings and body text

Keep the following unchanged:
- Company name: [name]
- Addresses and phone numbers
- Proper nouns and place names

Use professional, natural [target language] that fits a business in [industry].
Add a language switcher in the menu with flags for the available languages.
```

---

### Simple English button

<Tip>
  **Copy this prompt and paste it into Nettify AI:**
</Tip>

```
Add an "English" button at the top right of the navigation with a nice design.

The button should:
- Show the text "EN" or "English" with a small flag icon
- Have a subtle but visible design that matches the menu
- When clicked, all text on the current page should display in English
- A button for the original language should appear instead, so you can switch back

Translate the homepage to English with natural, professional language.
```

## Tips for multilingual websites

<CardGroup cols={2}>
  <Card title="Start with the homepage" icon="house">
    Translate the most important pages first — homepage, services and contact. You can always add more pages later.
  </Card>
  <Card title="Use flags or codes" icon="flag">
    Flag icons or simple codes like **EN / ES** make it easy for visitors to find the language option.
  </Card>
  <Card title="Double-check the translation" icon="spell-check">
    Nettify AI translates well, but it's always wise to review the text — especially industry terms and names that should be kept as-is.
  </Card>
  <Card title="Remember contact info" icon="address-card">
    Addresses, phone numbers and email are the same in all languages — but opening hours and greetings should be adapted.
  </Card>
</CardGroup>
