## How to replace images in the editor

The manual editor shows you all images in a selected element and lets you replace them individually.

<Steps>
  <Step title="Select the element containing the image">
    Click **"Select element"** and click on the element with the image you want to change. This can be the image itself, or a section that contains images.
  </Step>

  <Step title="Find the image in the image gallery">
    In the panel you'll see an **Images** section showing all images in the selected element. Each image appears as a small preview.
  </Step>

  <Step title="Replace the image">
    You have two options:
    - **Upload** — choose an image from your computer
    - **Paste URL** — enter the address of an image on the web
  </Step>

  <Step title="See the result and save">
    The new image appears immediately in the preview. Click **"Save changes"** when you're happy.
  </Step>
</Steps>

---

## Supported image types

The editor handles two types of images:

<CardGroup cols={2}>
  <Card title="Regular images" icon="image">
    Standard image elements that display an image directly (e.g. profile pictures, product photos, logos).
  </Card>
  <Card title="Background images" icon="panorama">
    Images used as backgrounds on sections and boxes (e.g. hero sections with a background photo).
  </Card>
</CardGroup>

The editor automatically detects which type it is and replaces the image correctly.
