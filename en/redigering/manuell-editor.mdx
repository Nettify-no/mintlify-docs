## What is the manual editor?

The manual editor gives you full control over every element on your website. While Nettify AI is perfect for big changes and new content, the manual editor is ideal for fine-tuning — like changing a color, adjusting a text, or replacing an image.

<Info>
  **Nettify AI + Manual editor = The perfect combination.** Use Nettify AI to
  build and make big changes, and the manual editor to fine-tune the details.
</Info>

---

## How to use the editor

<Steps>
  <Step title="Open the editor">
    Log in to [nettify.no](https://nettify.no) and select the website you want to edit. Click **"Edit page"** in the panel on the right.
  </Step>

  <Step title="Select element">
    Click the **"Select element"** button. This lets you click directly on elements in the website preview.

    <Tip>
      Click **"Deactivate"** to stop selecting elements.
    </Tip>

  </Step>

<Step title="Click on the element you want to change">
  Click on the text, image, button or section you want to edit. The element gets highlighted and **Element properties** appear in the right panel — with tools for content, appearance and images.
</Step>

<Step title="Make your changes">
  Use the tools in the panel to change text, colors, images, spacing and more. You see the changes in real-time in the preview.
</Step>

  <Step title="Save your changes">
    Click **"Save changes"** at the bottom of the panel when you're happy.

    <Warning>
      Made a mistake? Click **"Revert to published version"** to undo all changes.
    </Warning>

  </Step>
</Steps>

## Editor vs. Nettify AI

When should you use what? Here's a simple overview:

| Task                    | Nettify AI | Manual editor |
| ----------------------- | :--------: | :-----------: |
| Create new section      |     ✅     |               |
| Create new page         |     ✅     |               |
| Write lots of new text  |     ✅     |               |
| Change a heading        |     ✅     |      ✅       |
| Adjust a color          |            |      ✅       |
| Replace an image        |     ✅     |      ✅       |
| Fine-tune spacing       |            |      ✅       |
| Change font size        |            |      ✅       |
| Remove a section        |     ✅     |      ✅       |
| Change a link           |            |      ✅       |
